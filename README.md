# MONSTER CLEANUP

## 概要
本作品は、制限時間60秒の中で次々と出現する敵を倒し続けるアクションゲームです。プレイヤーは限られた時間の中で効率よく敵を処理し、より高いスコアを目指します。

## 操作方法
移動：WASDキー
攻撃：スペースキー
カメラ操作：方向キー（右左のみ）
回避：移動中に左シフトキー

## プレイ時間
1回のプレイにつき約1分

## 制作期間
4週間

## 制作人数
1人

## 担当箇所
プログラム全般

## 使用技術・ツール
・Unity(ver.6000.2.9f1)
・Audacity
・Adobe Express

## 工夫した点
・プレイヤーのコンボ攻撃の滑らかさや、ノックバック制御など、アニメーションにこだわりました。
・短時間でも爽快感を得られるよう、敵の出現間隔やテンポを調整しました。

## 苦労した点
・敵の当たり判定の制御に苦労しました。地面をすり抜けてしまう問題や、攻撃モーション中でないにも関わらず当たり判定が発生してしまう不具合があり、意図した挙動にならない場面が多くありました。そこで、当たり判定の有効・無効の切り替えタイミングや衝突判定の条件を見直し、動作を確認しながら調整を重ねることで、安定した挙動を実現しました。

## アピールポイント
限られた時間の中で次々と敵を倒していくテンポの良さと、繰り返し挑戦したくなるゲーム性にこだわりました。

## 補足
キーボードでの操作を想定しています。

ユーザーにとって分かりやすく、繰り返し遊びたくなる体験を意識して制作しました。
